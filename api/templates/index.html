Anonymizer REST API

Data Masking

METHOD Full

Description

The "full" method returns an anonymized dataset, transforming the database fields information into an asterisk.

Request body

                    
{
    "params": [
        {
            "tool": "masking",
            "fields": [
                "name",
                "surname"
            ]
        }
    ],
    "database": [
        {
            "name": "Wain",
            "surname": "Mitchard"
        },
        {
            "name": "Mariam",
            "surname": "Backson"
        },
        {
            "name": "Amalea",
            "surname": "Gambles"
        }
    ]
}

Response


{
    "database": [
        {
            "name": "*",
            "surname": "*"
        },
        {
            "name": "*",
            "surname": "*"
        },
        {
            "name": "*",
            "surname": "*"
        }
    ]
}
